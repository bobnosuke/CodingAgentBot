# 📜 CoderAgent コマンドリファレンス

CoderAgent は、**コマンド・サブコマンド数を最小限に抑え、Discord の UI 要素（Embed、Select Menu、Button、Modal）を最大限に活用する** 設計思想に基づいています。

---

## 🛠️ コマンド機能一覧

### 🚀 `/coding` コマンド (管理者専用)

| サブコマンド | 説明 | 備考 |
|:---|:---|:---|
| `/coding panel` | 開発用 Embed パネルと Select メニューを表示します。 | 開発開始、プロジェクト一覧、詳細確認、プロジェクト名変更が選択可能。パネルには説明と `Made by RovaexTeam` のフッターテキストが表示されます。選択肢は絵文字と簡易説明で視覚的に分かりやすく表示されます。 |
| `/coding server` | コマンド実行サーバー内で作業中のプロジェクト数、モデルの利用分布、その他便利なデータを表示します。 | サーバーの運用状況を把握できます。 |

### ⚙️ `/setting` コマンド

| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/setting` | ユーザー設定表示用 Embed パネルを表示します。 | どんな設定・確認ができるかを視覚的に分かりやすく表示し、その下に「設定を確認」ボタンを配置。押すと ephemeral で設定パネルが表示されます。 |
| `!setting` | ephemeral で設定パネルを直接表示します。 | 各種設定項目と現在の設定内容が書かれた Embed が表示されます。設定項目は Select メニューで選択し、絵文字と説明で視覚的に分かりやすく表示されます。 |

### 📚 `/guide` コマンド

| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/guide` | CoderAgentBot の使い方を Embed パネルで表示します。 | 各コマンドの説明や CodingRoom 内で行えることなどを記載。ボタンでページ送りが可能で、最初と最後のページではボタンがグレーアウトされます。 |
| `!guide` | `/guide` と同様の機能を提供します。 | |

### 🌐 `/config` コマンド (サーバー管理者専用)

| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/config` | サーバー単位の設定を Embed パネルで操作します。 | CodingRoom のカテゴリ、1ユーザーあたりの同時チャンネル数、サーバー全体の利用人数制限を設定可能。チャンネル数と人数制限は「無制限」も設定できます。 |

### 🩺 `/health` コマンド (Bot 運営オーナー専用)

| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/health` | DB 接続や API 状態などの健全性を確認できる Embed パネルを表示します。 | ボットの稼働状況を詳細に把握できます。 |

### 📈 `/stats` コマンド (Bot 運営オーナー専用)

| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/stats` | 全サーバーの利用統計を見やすく表示します。 | データ化可能な情報はデータグラフなどで視覚的に分かりやすく表示します。 |

### 🛑 `/shutdown` コマンド (Bot 運営オーナー専用)

| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/shutdown` | ボット本体を安全に停止します。 | 予期せぬ停止を防ぐためのオーナー専用コマンドです。 |

## 🏠 CodingRoom 内コマンド

CodingRoom が作成された際に、以下のコマンドの説明が Embed で必ず表示されます。

| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `!list` | プロジェクト内のファイルを階層構造で分かりやすく表示します。 | |
| `!get` | 実行後に Select メニューで表示するファイルを選択させ、選んだファイルをコードブロックで表示します。 | Discord の送信文字数制限を上回る場合はファイルで送信されます。 |
| `!download` | プロジェクト内のファイル一覧を Select メニューで表示します。 | 複数選択を可能にし、ダウンロードするファイルを選ばせます。選んだファイルを ZIP にまとめて送信。Select の下に「全てダウンロード」ボタンも配置。プロジェクト内にファイルが1つしかない場合は、実行後そのままファイルを送信（ZIP にまとめる必要はありません）。 |
| `!close` | 警告メッセージと「はい」「いいえ」ボタンを設置します。 | 「いいえ」が押されたら警告メッセージとボタンを削除。「はい」が押されたら CodingRoom とセッション情報（その CodingRoom のキャッシュなど）を全て完全に削除します。 |
| `!readme` | プロジェクトの README を表示します。 | |

## 💬 CodingRoom 内での AI 対話

CodingRoom 内では、`!` で始まるコマンド以外のユーザー発言（AI 利用ユーザーのみ。管理者などが発言した場合は無視）は、すべて AI との会話と認識されます。これにより、`/coding chat` コマンドは不要となります。
