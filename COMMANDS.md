# 📜 CoderAgent コマンドリファレンス

CoderAgent は、**スラッシュコマンド (`/`)** と **プレフィックスコマンド (`!`)** の両方に対応したハイブリッド形式を採用しています。

---

## 🛠️ 一般ユーザー用コマンド

### 🔑 API キー管理
| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/api-key register` | OpenRouter API キーを登録します。 | 入力は暗号化されます。 |
| `/api-key remove` | 登録済みの API キーを削除します。 | 復旧はできません。 |

### 🚀 コーディング機能
| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/coding start [project_name]` | 新しいコーディングセッションを開始します。 | プライベートチャンネルが作成されます。 |
| `/coding chat <message>` | AI と対話してコードを生成・修正します。 | **CodingRoom 内限定** |
| `/coding end` | セッションを終了し、チャンネルを削除します。 | **CodingRoom 内限定** |
| `/coding panel` | UI (Select Menu) 形式の操作パネルを表示します。 | スラッシュコマンドが苦手な方向け。 |
| `/coding list` | 自分のプロジェクト一覧を表示します。 | 過去のプロジェクトも確認可能。 |
| `/coding info` | 現在のセッションの詳細情報を表示します。 | 使用モデル、ファイル数など。 |
| `/coding export` | プロジェクトを ZIP 形式で書き出します。 | **CodingRoom 内限定** |
| `/coding rename <new_name>` | プロジェクト名を変更します。 | **CodingRoom 内限定** |
| `/coding delete` | プロジェクトデータを完全に削除します。 | 確認画面が表示されます。 |

### 📁 ファイル管理 (CodingRoom 内限定)
| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/save <filename> <content>` | 指定したファイル名でコードを保存します。 | |
| `/list` | 保存されているファイル一覧を表示します。 | |
| `/get <filename>` | 指定したファイルの内容を表示します。 | |
| `/download` | 全ファイルを ZIP でダウンロードします。 | `/coding export` と同等。 |

### ⚙️ 設定・情報
| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/setting` | ユーザー設定パネルを表示します。 | モデル変更、利用状況確認など。 |
| `/guide [topic]` | CoderAgent の使い方ガイドを表示します。 | |
| `/about` | ボットの情報を表示します。 | |
| `/status` | ボットの稼働状況を表示します。 | |

---

## 🛡️ 管理者・オーナー用コマンド

### 🌐 サーバー管理者用 (`/config`)
| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/config category <id>` | CodingRoom を作成するカテゴリを指定します。 | |
| `/config limit_user <num>` | 1ユーザーあたりの同時チャンネル数を制限します。 | |
| `/config limit_server <num>` | サーバー全体の利用人数を制限します。 | |

### 👑 ボットオーナー専用
| コマンド | 説明 | 備考 |
|:---|:---|:---|
| `/health` | DB 接続や API 状態などの健全性を確認します。 | |
| `/stats` | 全サーバーの利用統計を表示します。 | |
| `/shutdown` | ボットを安全に停止します。 | |

---

## 💡 プレフィックスコマンドの利用
スラッシュコマンドが利用できない場合は、`!` を頭に付けて実行してください。
例: `!coding start`, `!api-key register`
