1. プロジェクト概要
1.1 プロジェクト名
正式名称：
CoderAgent
※名称は仮称。正式公開時に変更可能。
 
⸻
 
1.2 プロジェクト概要
CoderAgentは、Discord上で利用可能なAIコード開発エージェントBotである。
CLIベースで提供されているClaude Codeのような開発体験を、Discord Bot形式へ移植する。
ユーザーはDiscordのチャット上でAIエージェントと対話することで、
* コード生成
* コード修正
* ファイル作成
* プロジェクト構築
* 開発相談
などを実行できる。
従来CLI操作が必要だったAI開発環境をDiscordへ提供することで、PCを持たないスマートフォンユーザーでも利用可能にする。
 
⸻
 
2. 開発背景
2.1 現在存在する課題
現在、高性能AIコードエージェントを利用する場合、以下の問題が存在する。
問題1
API料金
高性能モデルを利用する場合、API利用料金が高額になる。
個人開発者や初心者にとって継続利用の障壁となる。
 
⸻
 
問題2
CLI操作が必要
Claude Codeなどの開発エージェントはCLI操作を前提としている。
そのため、
* ターミナル環境
* PC
* コマンド操作知識
が必要になる。
スマートフォンユーザーやプログラミング初心者には利用ハードルが高い。
 
⸻
 
問題3
AI開発環境構築の難しさ
AIを利用した開発では、
* API設定
* Python環境構築
* Node環境構築
* CLI設定
* ファイル管理
など複数の準備が必要。
CoderAgentではDiscordだけで利用できる環境を提供する。
 
⸻

153AIエージェントとの対話（自律パイプライン）
154ユーザーが自然言語で要求を入力。
処理フロー（2段階自律パイプライン）：
1. 要件定義フェーズ (Gemini 1.5 Flash利用)
   - ユーザーの曖昧な要望を分析し、技術仕様・実装計画をJSON化。
   - ユーザーへ「要件確認パネル」を提示。
2. 実装フェーズ (OpenRouter 高性能モデル利用)
   - ユーザーの承認後、JSONに基づき一括でコードを生成。
   - 1回のリクエストで全ファイルを完成させる。

メリット：
- APIリクエスト制限（1日50回等）の回避。
- ユーザーとの合意形成後の実装による、開発精度の向上。
- プログラミング未経験者でも利用可能な「要件の自動整理」。

7. 技術選定
7.1 開発言語
採用候補
Python 3.12
 
⸻
 
理由
* Discord Bot開発実績が豊富
* AI APIとの連携が容易
* 非同期処理に対応
* 将来的な機械学習・AI連携との相性が良い
 
⸻
 
7.2 Discordライブラリ
採用
discord.py
 
⸻
 
理由
* 安定性
* 活発な開発
* Slash Command対応
* Cog構成対応
* ドキュメント量
 
⸻
 
8. 外部AI API
採用
OpenRouter API
 
⸻
 
理由
複数AIモデルを統一API形式で利用できるため。
将来的に
* Claude
* GPT
* Gemini
* その他LLM
への切り替えが容易。

| `/health` | - | Bot運営オーナー専用 | DB 接続や API 状態などの健全性を確認できる Embed パネルを表示します。 |
| `/stats` | - | Bot運営オーナー専用 | 全サーバーの利用統計を見やすく表示します。 |
| `/shutdown` | - | Bot運営オーナー専用 | ボット本体を安全に停止します。 |
| `!list` | - | CodingRoom内 | プロジェクト内のファイルを階層構造で分かりやすく表示します。 |
| `!get` | - | CodingRoom内 | 実行後に Select メニューで表示するファイルを選択させ、選んだファイルをコードブロックで表示します。 |
| `!download` | - | CodingRoom内 | プロジェクト内のファイル一覧を Select メニューで表示し、選択したファイルをZIPにまとめて送信します。 |
| `!close` | - | CodingRoom内 | CodingRoom とセッション情報を全て完全に削除します。 |
| `!readme` | - | CodingRoom内 | プロジェクトの README を表示します。 |
