# 📝 CoderAgent 共同作業タスクリスト

このファイルは、CoderAgent プロジェクトにおける共同作業のタスク管理に使用します。
各エージェントは、作業を開始する前にこのファイルを確認し、自身の担当タスクを明確にしてください。

## 1. 共同作業フロー

1.  **作業開始前**: `git pull origin main` で最新のコードを取得し、この `TODO_COLLAB.md` を確認して担当タスクを把握します。
2.  **ブランチ作成**: 各タスクは新しいブランチで作業します (`git checkout -b feature/your-task-name`)。
3.  **実装**: コードを実装します。
4.  **コミット**: 変更内容をコミットします (`git commit -m "feat: [タスク内容の簡潔な説明]"`)。
5.  **プッシュ**: 作業が完了したら、ブランチをリモートにプッシュします (`git push origin feature/your-task-name`)。
6.  **プルリクエスト**: `main` ブランチへのプルリクエストを作成します。
7.  **タスク更新**: 自身の担当タスクのステータスを更新し、完了したタスクは `✅` に変更します。

## 2. コミュニケーションルール

- 各エージェントは、自身の発言の前に必ず**独自の名前**を記載してください。
- 会話が長くなった場合、自身の発言箇所がある程度の文量に達したら、その部分を削除し、新しいセクションとして書き始めてください。
- 他のエージェントへの依頼や確認事項は、明確に相手を指名して記載してください。
- 要件定義書や `TODO_COLLAB.md` の内容に変更があった場合は、`COMMUNICATION.md` で共有し、関連ファイルを修正した上でGitHubにプッシュしてください。
